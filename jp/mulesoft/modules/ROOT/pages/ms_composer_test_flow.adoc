= コンポーザフローのテスト

フローの作成中にテストすることで、アクティベーション前にフローの各ステップと各項目のマッピングが正常に機能することを確認できます。頻繁に検証することで、フローの効果に関するほぼリアルタイムのフィードバックが得られ、作成中にエラーを修正できるようになります。

テストでは、1 つのレコードを処理してテストします。すべてのレコードを処理するにはフローをアクティブ化する必要があります。

フローをテストするときは、次の点に注意してください。

* トリガステップが必須です。
* テストは、エラーで停止するか手動で終了しなかった場合、10 分後に自動的に終了します。
* フロー全体が実行され、テストされます。

フローをテストするときは、次の手順を実行します。

* 無効なアカウントを使用して、Sandbox 環境でフローを作成してフローをテストします。
* サンプルまたはダミーレコードを使用してフローをテストします。
* テストがタイムアウトになる場合は、処理するレコード数を制限することを検討してください。

フローをテストする手順は、次のとおりです。

. コンポーザのフローリストで、テストするフローをクリックして ​*[Edit (編集)]*​ をクリックします。
. *[Test (テスト)]*​ をクリックします。 +
テストが完了すると、コンポーザによって通知されます。テストの実行結果がカードに表示されます。
